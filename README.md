# 👩‍💻 Meu Portfólio Pessoal

Bem-vindo(a) ao repositório do meu portfólio profissional! Este projeto foi desenvolvido para centralizar minhas informações, apresentar minhas habilidades técnicas na área de desenvolvimento Front-End, exibir meus principais projetos e listar minhas certificações.

---

## 📱 Desenvolvido Inteiramente pelo Celular

Um dos maiores diferenciais deste projeto está no seu processo de construção. Por não possuir um computador pessoal, transformei o desafio em motivação: **toda a codificação, estilização, correção de bugs e gerenciamento de versão via Git foram realizados diretamente de um dispositivo móvel**.

### 🛠️ Ferramentas Utilizadas no Mobile:
* **Acode**: Editor de código mobile utilizado para escrever o HTML, CSS e JavaScript.
* **Termux**: Emulador de terminal Linux para Android utilizado para executar comandos Git e gerenciar os repositórios.

---

## 🚀 Tecnologias e Recursos Utilizados

* **HTML5 Semântico**: Estruturação limpa e acessível.
* **CSS3 Moderno**: Layout responsivo criado para funcionar perfeitamente em telas de computadores e smartphones.
* **JavaScript (ES6+)**: Manipulação dinâmica do DOM para as funcionalidades interativas.
* **Design Customizado**:
  * Alternador de tema dinâmico (Modo Claro ☀️ / Modo Escuro 🌙) com correções automáticas de contraste.
  * Cursor universal totalmente personalizado integrado via CSS para navegação em desktops.

---

## 🎨 Funcionalidades em Destaque

* **Responsividade Completa**: Interface totalmente adaptável para qualquer tamanho de tela.
* **Dark/Light Mode**: Sistema de troca de temas que preserva a legibilidade de todos os cards, títulos e links do menu.
* **Botão Voltar ao Topo**: Navegação otimizada para melhorar a experiência do usuário (UX).
